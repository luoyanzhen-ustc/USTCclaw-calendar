## 功能详解

### 1. 课表导入

**步骤：**

1. 上传课表图片（截图、照片均可）
2. 助手自动识别并展示结果
3. 确认无误后回复"好的"
4. 课程自动添加到日历

**示例对话：**

```
[用户上传课表图片]

助手：📚 识别到你的课表啦！我整理了一下：

**周一**
- 07:50-09:25  信息论 A @ GT-B210
- 09:45-11:20  数学分析 (B2) @ 3C202
- 14:00-15:35  统计信号分析与处理 @ GH-408

要帮你把这些课程都记到日历里吗？

用户：好的

助手：✅ 已添加 12 门课程！现在可以问我"今天有什么课"啦~
```

### 2. 查询日程

**今日日程：**
```
今天有什么课？
今天的安排
```

**明日日程：**
```
明天呢？
明天有什么课？
后天下午有事吗
```

**本周日程：**
```
这周安排
本周课表
下周有什么课
```

**特定日期：**
```
下周三下午有课吗
周五全天安排
```

### 3. 添加事件

**简单添加：**
```
明晚 7 点去图书馆
后天下午 3 点和朋友出去玩
```

**详细添加：**
```
下周一上午 10 点在实验室开会，提前 30 分钟提醒我
```

**助手会询问：**
- 确认时间
- 确认地点
- 设置提醒时间

### 4. 修改事件

**修改时间：**
```
把明天的图书馆改成 8 点
组会改到下午 4 点
```

**修改地点：**
```
图书馆的地点改成 3 楼自习室
```

**修改提醒：**
```
提前 1 小时提醒我
不用提醒了
```

### 5. 删除事件

```
取消明天的图书馆计划
删除后天的聚会
不要提醒我了
```

### 6. 智能提醒

**自动提醒：**
- 高优先级事件：提前 1 天 + 1 小时
- 普通课程：提前 30 分钟
- 低优先级：提前 10 分钟

**多渠道推送：**
- 当前聊天（默认）
- QQ（需配置）
- 微信（需配置）

**静默时段：**
- 默认 23:00-08:00 不推送
- 可在设置中修改

## 高级功能

### 优先级设置

**高优先级（⚡）：**
- 考试、答辩
- 重要会议
- 截止日期

**中优先级（📌）：**
- 普通课程
- 组会、例会

**低优先级（📝）：**
- 日常计划
- 休闲活动

### 时间表达

助手支持多种时间表达：

| 表达 | 含义 |
|------|------|
| 今天、今天 | 今天 |
| 明天、明晚 | 明天 |
| 后天 | 后天 |
| 下周一 | 下周一 |
| 早上、上午 | 上午时段 |
| 下午 | 下午时段 |
| 晚上、今晚 | 晚上时段 |
| 7 点、晚上 7 点 | 具体时间 |

### 上下文理解

助手支持多轮对话：

```
用户：明天有什么课？
助手：（展示明天课表）

用户：那后天呢？
助手：（展示后天课表）

用户：大后天呢？
助手：（展示大后天课表）
```

## 常见问题

### Q: 如何修改默认提醒时间？

A: 说"设置默认提醒为 30 分钟"即可。

### Q: 可以关闭提醒吗？

A: 说"关闭所有提醒"或"静默模式"。

### Q: 如何导出课表？

A: 说"导出课表"或"生成 ICS 文件"（开发中）。

### Q: 支持共享日历吗？

A: 开发中，可以期待后续版本！
