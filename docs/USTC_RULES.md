# USTC 课表识别规则

## 节次时间映射

USTC 固定节次时间如下：

| 节次 | 时间 | 说明 |
|------|------|------|
| 1-2 | 07:50-09:30 | 早课 |
| 3-5 | 09:45-12:10 | 上午主课 |
| 6-7 | 14:00-15:40 | 下午课前段 |
| 8-10 | 15:55-18:30 | 下午课完整 |
| 11-13 | 18:40-21:55 | 晚课 |

## 课表格式

### 研究生课表

格式：`课程名称 地点 (周次) 星期 (节次)`

示例：
```
机器学习系统 G2-B403 (2~15 周) 2(3,4,5)
```

解析：
- 课程名称：机器学习系统
- 地点：G2-B403
- 周次：2~15 周
- 星期：2（周二）
- 节次：3,4,5（第 3-5 节）

### 本科生课表

格式：`课程名称 教师 地点 (周次) 星期 (节次)`

示例：
```
高等数学 张娜 3C202 (1-18 周) 1(3,4)
```

解析：
- 课程名称：高等数学
- 教师：张娜
- 地点：3C202
- 周次：1-18 周
- 星期：1（周一）
- 节次：3,4（第 3-4 节）

## 识别规则

### 1. 自动检测课表类型

```javascript
function detectScheduleType(text) {
  if (text.includes('教师') || text.includes('任课教师')) {
    return 'undergraduate'; // 本科生课表
  } else {
    return 'graduate'; // 研究生课表
  }
}
```

### 2. 周次范围解析

支持多种格式：
- `1~15` → [[1, 15]]
- `1-15` → [[1, 15]]
- `1~5, 7~14` → [[1, 5], [7, 14]]
- `1, 3, 5` → [[1, 1], [3, 3], [5, 5]]

### 3. 节次时间计算

```javascript
function calculateTime(periods) {
  const startPeriod = periods[0];
  const endPeriod = periods[periods.length - 1];
  
  const startTime = USTC_PERIOD_TIMES[startPeriod][0];
  const endTime = USTC_PERIOD_TIMES[endPeriod][1];
  
  return { startTime, endTime };
}
```

## 特殊课程处理

### 晚课（18:40 开始）

部分研究生课程从 18:40 开始：
- 工程伦理
- 人工智能与分子科学

这些课程标记为 `specialTime: true`，提醒时间调整为提前 45 分钟。

### 跨节次课程

如"信号完整性分析"（6-9 节，14:00-17:30）：
- 自动计算总时长
- 设置合适的提醒时间

## 识别准确率优化

### 已知问题

1. **合并单元格**：同一时间多门课程可能识别错误
   - 解决：后处理阶段检测冲突

2. **周次范围复杂**：如"1~5, 7~14, 16~18"
   - 解决：正则表达式解析多个范围

3. **图片质量**：模糊、反光导致识别错误
   - 解决：提示用户重新上传
