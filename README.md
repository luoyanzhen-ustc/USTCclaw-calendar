# 📅 USTCclaw Calendar

**OpenClaw 智能日历助手** - 专为 USTC 学生设计的课表管理与日程提醒工具

> 🌸 自然语言交互 · 课表图片识别 · 多渠道提醒 · 数据本地存储

## ✨ 功能亮点

- 🖼️ **课表图片识别** - 直接上传课表截图，自动识别并导入
- 🗣️ **自然语言交互** - 无需命令，像聊天一样管理日程
- 📆 **智能视图** - 日视图/周视图，一目了然
- ⏰ **多渠道提醒** - QQ/微信/当前聊天，重要事情不错过
- 🔒 **隐私保护** - 所有数据存储在本地，不上传云端
- 🎯 **USTC 专属** - 适配中国科学技术大学课表格式

## 🚀 快速开始

### 前置要求

- OpenClaw 已安装并运行
- 已连接 QQ 或 微信（用于提醒推送）
- 模型支持图片识别（qwen-chat 或类似）

### 安装步骤

1. **克隆或下载本仓库**

```bash
git clone https://github.com/luoyanzhen-ustc/USTCclaw-calendar.git
cd USTCclaw-calendar
```

2. **复制 Skill 到 OpenClaw 工作区**

```bash
cp -r skill ~/.openclaw/workspace/skills/calendar-assistant
```

3. **重启 OpenClaw（如需要）**

```bash
openclaw gateway restart
```

### 初始化

在聊天中直接上传你的**课表图片**，助手会自动识别并引导你完成初始化！

或者输入：
```
帮我初始化日历
```

## 📖 使用指南

### 查看日程

```
今天有什么课？
明天呢？
这周安排
下周三下午有空吗
```

### 添加计划

```
明晚 7 点去图书馆
后天下午 3 点和朋友出去玩
下周一上午记得交作业
```

### 管理事件

```
取消明天的图书馆计划
把组会时间改到下午 4 点
提醒我提前 1 小时
```

### 导入课表

直接**上传课表图片**（截图、照片均可），助手会自动识别并询问是否添加。

## 📋 示例对话

### 课表导入

```
[用户上传课表图片]

助手：📚 识别到你的课表啦！我整理了一下：

**周一**
- 07:50-09:25  信息论 A @ GT-B210
- 09:45-11:20  数学分析 (B2) @ 3C202
- 14:00-15:35  统计信号分析与处理 @ GH-408

要帮你把这些课程都记到日历里吗？

用户：好的

助手：✅ 已添加 12 门课程！现在可以问我"今天有什么课"啦~
```

### 日常查询

```
用户：今天下午有课吗

助手：📅 今天 4 月 7 日 周一

14:00 📝 统计信号分析与处理
      📍 GH-408
      👤 叶中付、徐旭
      ⏰ 13:30 提醒你

这是今天最后一节课，上完就可以休息啦~
```

### 添加临时计划

```
用户：明晚 7 点我要去图书馆

助手：好的！记下来：
      📝 明晚 19:00 图书馆学习
      ⏰ 需要提前多久提醒你？（15 分钟/30 分钟/1 小时/不用）

用户：30 分钟

助手：✅ 已添加！明晚 18:30 我会提醒你~
```
