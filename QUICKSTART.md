# 🚀 快速开始

欢迎使用 USTCclaw Calendar！只需 3 步即可开始使用。

## 第 1 步：部署

在项目目录下运行：

```bash
bash scripts/deploy.sh
```

这会自动：
- 复制 Skill 到 OpenClaw
- 创建配置文件
- 初始化日历目录

## 第 2 步：重启 OpenClaw（如需要）

```bash
openclaw gateway restart
```

## 第 3 步：开始使用

### 方式 A：上传课表图片（推荐）

1. 在聊天中上传你的**课表图片**
2. 助手会自动识别并展示结果
3. 回复"好的"确认添加

### 方式 B：手动初始化

在聊天中输入：
```
帮我初始化日历
```
